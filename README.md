# 🌿 Spring Visual Builder

A visual low-code tool for generating Spring Boot project structure.
Instead of writing boilerplate code manually, simply use your mouse
to design Entities, Relationships, Services and Controllers —
and get production-ready Java code generated automatically.

## 🚧 Status
Currently under active development...

## 💡 Idea
Inspired by the gap between visual tools like Adobe Acrobat & Photoshop
and the lack of visual tooling for Spring Boot developers.

## 🎯 Planned Features
- [ ] Visual Entity Designer (add columns, types, constraints)
- [ ] Relationship Mapper (@OneToMany, @ManyToOne, @ManyToMany)
- [ ] Service Class Generator
- [ ] REST Controller Generator
- [ ] Full Spring Boot project export as ZIP

## 🛠️ Tech Stack
- **Backend:** Java, Spring Boot
- **Frontend:** React.js (planned)
- **Template Engine:** Apache FreeMarker (planned)
